# ChessTerm SDK

ChessTerm SDK 是使用各编程语言对 [ChessTerm API](./api.md) 的封装，将来可能还会添加更多编程语言的支持。

## Python

ChessTerm SDK for Python 使用 [`requests` 库](https://requests.readthedocs.io/zh_CN/latest/) 实现了 ChessTerm 的 [RESTful API](./api.md#2)，将来可能还会添加 [Socket.io API](./api.md#3) 的实现。

 - Install via `pip install chessterm-sdk`.
 - [GitHub Repo](https://github.com/JingBh/chessterm-sdk-python)
 - [PyPI Project](https://pypi.org/project/chessterm-sdk)
